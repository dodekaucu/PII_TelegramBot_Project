 Public Interface con la firma de los metodos que debe poseer un Usuario. Se utilizo una interface puesto que todos los usuarios deben tener almenos ciertas properties en orden de ser considerados un Usuario. Se cumple el principio OCP, puesto que al crear una interface usuario permito que las clases que dependan de IUsuario, queden abiertas a la extension pero cerradas a la modificacion.
