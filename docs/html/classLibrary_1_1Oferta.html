Detailed Description

Esta clase representa una oferta. Es experta (EXPERT) en la creación de ofertas. Ademas se utiliza el patron CREATOR, pues los materiales son creados al momento de inicializar la oferta. Pues oferta usa de forma muy cercana los materiales. => oferta tiene la responsabilidad de crear materiales.
